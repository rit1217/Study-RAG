# รูปแบบผลลัพธ์
เสนอการปรับปรุงเป็นภาษาไทย ในรูปแบบ:

## 1. สรุปปัญหาที่พบ
- ระบุจุดอ่อนของ prompt ปัจจุบันที่ทำให้คะแนนไม่ตรงกับมนุษย์

## 2. ข้อเสนอแก้ไขเฉพาะจุด
สำหรับแต่ละจุดที่ต้องแก้ไข ระบุ:
- **ส่วนที่ต้องแก้:** ข้อความเดิมใน prompt
- **ข้อความใหม่ที่เสนอ:** ข้อความที่แนะนำให้ใช้แทน
- **เหตุผล:** ทำไมการแก้ไขนี้จะช่วยให้คะแนนสอดคล้องกับมนุษย์มากขึ้น

## 3. เกณฑ์ที่ควรเพิ่ม/ปรับ
- เกณฑ์การประเมินที่ขาดหายหรือไม่ชัดเจนใน prompt ปัจจุบัน

## 4. ผลกระทบที่คาดหวัง
- คาดว่าการปรับปรุงจะช่วยให้คะแนนสอดคล้องกับมนุษย์มากขึ้นอย่างไร
