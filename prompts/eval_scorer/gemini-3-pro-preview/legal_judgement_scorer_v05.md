# งานของคุณ
ให้คะแนนความถูกต้องของ **ข้อ 2 (คำวินิจฉัย)** ในคำตอบ (OUTPUT) โดยเทียบกับเฉลย (EXPECTED)

# รูปแบบข้อมูลเฉลย
เฉลย (EXPECTED) อยู่ในรูปแบบ Markdown ที่แบ่งเป็นหัวข้อ:
- `## 1. มาตรากฎหมายที่เกี่ยวข้อง` — มาตรากฎหมายและคำพิพากษาศาลฎีกาที่เกี่ยวข้อง
- `## 2. คำวินิจฉัย` — คำวินิจฉัยที่ถูกต้อง (ใช้เป็นเฉลยหลักสำหรับการประเมินนี้)
- `## 3. ข้อสรุปและข้อเสนอแนะ` — ข้อสรุปและข้อเสนอแนะ (อาจไม่มีในบางข้อ)

ให้ประเมิน OUTPUT โดยเทียบกับ **ข้อ 2 (คำวินิจฉัย)** ในเฉลย เป็นหลัก และใช้ข้อ 1 ประกอบเพื่อตรวจสอบว่าอ้างอิงมาตรากฎหมายถูกต้องหรือไม่

# เกณฑ์การประเมิน
1. **ความสอดคล้องกับธงคำตอบ (Bank Policy/Risk Appetite):**
   - ต้องยึดถือบทสรุปใน EXPECTED เป็นที่สุด หาก EXPECTED ระบุว่า "ทำไม่ได้" หรือ "มีความเสี่ยงสูง" แต่ OUTPUT ตอบว่า "ทำได้ตามทฤษฎี" ให้ถือว่า **ผิด (0.0)** ทันที

2. **ฐานกฎหมายที่ถูกต้อง (Legal Basis):**
   - ตรวจสอบว่าเหตุผลทางกฎหมายต้องตรงกับ EXPECTED ไม่ใช่แค่ผลลัพธ์สุดท้ายตรงกัน (เช่น ต้องอ้างเหตุผลเรื่อง "การผ่อนเวลา" ไม่ใช่ "การแปลงหนี้ใหม่") หากเหตุผลผิดแม้ผลลัพธ์ถูก ให้ถือว่าไม่ถูกต้อง

3. **ความละเอียดของเงื่อนไข (Granularity):**
   - ตรวจสอบว่า OUTPUT แยกแยะกรณีต่างๆ ได้ครบถ้วนตาม EXPECTED หรือไม่ (เช่น การแยกพิจารณากรณีลูกหนี้มีแผนฟื้นฟู vs ไม่มีแผนฟื้นฟู)

4. **ความถูกต้องทั่วไป:**
   - ผลทางกฎหมายที่วินิจฉัย (สมบูรณ์/โมฆะ/โมฆียะ) ต้องตรงกับเฉลย

# ผลการประเมิน
- ตอบได้ถูกต้อง ชัดเจน ครอบคลุม
- ไม่ถูกต้องตามเฉลย
- ตอบได้ถูกต้อง แต่ไม่ครอบคลุม มีเนื้อหาสำคัญตกหล่น
