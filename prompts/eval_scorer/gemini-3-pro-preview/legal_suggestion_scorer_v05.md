# งานของคุณ
ให้คะแนนความถูกต้องของ **ข้อ 3 (ข้อสรุปและข้อเสนอแนะ)** ในคำตอบ (OUTPUT) โดยเทียบกับเฉลย (EXPECTED)

# รูปแบบข้อมูลเฉลย
เฉลย (EXPECTED) อยู่ในรูปแบบ Markdown ที่แบ่งเป็นหัวข้อ:
- `## 1. มาตรากฎหมายที่เกี่ยวข้อง` — มาตรากฎหมายและคำพิพากษาศาลฎีกาที่เกี่ยวข้อง
- `## 2. คำวินิจฉัย` — คำวินิจฉัยที่ถูกต้อง
- `## 3. ข้อสรุปและข้อเสนอแนะ` — ข้อสรุปและข้อเสนอแนะ (ใช้เป็นเฉลยหลักสำหรับการประเมินนี้ หากไม่มีให้ใช้ข้อ 2 ประกอบ)

ให้ประเมิน OUTPUT โดยเทียบกับ **ข้อ 3 (ข้อสรุปและข้อเสนอแนะ)** ในเฉลย เป็นหลัก

# เกณฑ์การประเมิน
1. **ความสอดคล้องกับนโยบายและคำวินิจฉัย:**
   - หากคำวินิจฉัย (Judgement) ใน OUTPUT ผิดพลาด หรือขัดแย้งกับนโยบายใน EXPECTED ข้อเสนอแนะที่ตามมาควรได้คะแนนต่ำหรือศูนย์ทันที (ไม่ควรให้คะแนนแยกต่างหากในกรณีที่คำแนะนำนั้นตั้งอยู่บนสมมติฐานที่ผิด)

2. **ความจำเพาะเจาะจง (Keyword Matching):**
   - ตรวจสอบคำสำคัญ (Keywords) ที่ปรากฏใน EXPECTED เช่น ชื่อเอกสารเฉพาะ, ชื่อหน่วยงาน, หรือขั้นตอนการปฏิบัติงานที่ชัดเจน 
   - หาก OUTPUT ให้คำแนะนำกว้างๆ (General Advice) โดยขาดคำสำคัญเหล่านี้ ให้ถือว่า **ไม่ครอบคลุม**

3. **ความเป็นไปได้และการปฏิบัติจริง:**
   - ตรวจสอบว่าข้อเสนอแนะมีความเป็นรูปธรรม นำไปปฏิบัติได้จริง และครอบคลุมข้อควรระวัง/ความเสี่ยงตามเฉลย

# ผลการประเมิน
- ตอบได้ถูกต้อง ชัดเจน ครอบคลุม ให้ข้อเสนอแนะที่เป็นไปได้จริง
- ไม่ถูกต้องตามเฉลย
- ตอบได้ถูกต้อง แต่ไม่ครอบคลุม มีเนื้อหาสำคัญตกหล่น
