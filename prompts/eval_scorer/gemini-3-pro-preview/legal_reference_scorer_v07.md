# รูปแบบข้อมูลเฉลย
เฉลย (EXPECTED) อยู่ในรูปแบบ Markdown ที่แบ่งเป็นหัวข้อ:
- `## 1. มาตรากฎหมายที่เกี่ยวข้อง` — มาตรากฎหมายและคำพิพากษาศาลฎีกาที่เกี่ยวข้อง (ใช้เป็นเฉลยหลักสำหรับการประเมินนี้)

# เกณฑ์การประเมิน (Strict Verbatim & Formatting Check)
1. **ความถูกต้องระดับตัวอักษร:** ตรวจสอบเทียบกับเฉลยแบบคำต่อคำ (Word-for-word comparison) ตัวบทกฎหมายต้อง **ครบถ้วนทุกตัวอักษร** 
   - ห้ามมีการตัดทอน (Truncation)
   - ห้ามใช้เครื่องหมายละความ (...)
   - ห้ามสรุปความเอง (Paraphrasing)
   - ถ้อยคำเฉพาะทาง (Wording) เช่น 'คู่สมรส' vs 'สามีภริยา' ต้องตรงตามฉบับกฎหมายที่อ้างอิงในเฉลยเป๊ะๆ
2. **ความถูกต้องของมาตรา:** เลขมาตราและชื่อกฎหมายต้องถูกต้องตรงกับเฉลย
3. **สิ่งเจือปน (Noise):** ห้ามมีการอ้างอิงมาตราที่ไม่เกี่ยวข้อง (Irrelevant) ปะปนเข้ามา หากพบให้ถือว่าไม่ถูกต้องตามเกณฑ์
4. **รูปแบบ (Formatting):** การจัดย่อหน้าและรูปแบบต้องสอดคล้องกับต้นฉบับกฎหมาย

# ผลการประเมิน
เลือกผลลัพธ์เพียงข้อเดียวที่ตรงที่สุด:
- "อ้างอิงมาตรากฎหมายถูกต้องและครอบคลุมตามเฉลย": (1.0 คะแนน) เมื่อตัวบทครบถ้วนทุกตัวอักษร ตรงตามเฉลยแบบคำต่อคำ ไม่มีตัดทอน ไม่มีสรุปความ และไม่มีมาตราที่ไม่เกี่ยวข้องปะปน
- "อ้างอิงมาตรากฎหมายไม่ถูกต้อง หรือไม่เกี่ยวข้อง": (0.0 คะแนน) เมื่ออ้างอิงผิดมาตรา, มีมาตราที่ไม่เกี่ยวข้องปะปนมา, หรือเนื้อหาตัวบทผิดเพี้ยนไปจากกฎหมายจริง
- "อ้างอิงมาตรากฎหมายถูกต้อง แต่ไม่ครอบคลุม มีมาตราสำคัญตกหล่น": (0.5 คะแนน) เมื่อเลขมาตราถูก แต่ตัวบทมีการตัดทอน ใช้เครื่องหมาย (...) หรือสรุปความเอง หรือขาดวรรคตอนสำคัญ
