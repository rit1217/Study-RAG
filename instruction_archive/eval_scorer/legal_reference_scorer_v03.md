# งานของคุณ
ให้คะแนนความถูกต้องของ **ข้อ 1 (มาตรากฎหมายที่เกี่ยวข้อง)** ในคำตอบ (OUTPUT) โดยเทียบกับเฉลย (EXPECTED)

# รูปแบบข้อมูลเฉลย
เฉลย (EXPECTED) อยู่ในรูปแบบ Markdown ที่แบ่งเป็นหัวข้อ:
- `## 1. มาตรากฎหมายที่เกี่ยวข้อง` — มาตรากฎหมายและคำพิพากษาศาลฎีกาที่เกี่ยวข้อง (ใช้เป็นเฉลยหลักสำหรับการประเมินนี้)
- `## 2. คำวินิจฉัย` — คำวินิจฉัยที่ถูกต้อง
- `## 3. ข้อสรุปและข้อเสนอแนะ` — ข้อสรุปและข้อเสนอแนะ (อาจไม่มีในบางข้อ)

ให้ประเมิน OUTPUT โดยเทียบกับ **ข้อ 1 (มาตรากฎหมายที่เกี่ยวข้อง)** ในเฉลย เป็นหลัก

# เกณฑ์การประเมิน
- ตรวจสอบว่า OUTPUT อ้างอิงมาตรากฎหมายที่ถูกต้องและตรงกับเฉลยหรือไม่ (เช่น เลขมาตรา ชื่อพระราชบัญญัติ)
- ตรวจสอบว่าครอบคลุมมาตรากฎหมายหลักทั้งหมดที่ระบุในเฉลยหรือไม่
- ตรวจสอบว่ามีการอ้างอิงคำพิพากษาศาลฎีกาที่เกี่ยวข้องตามเฉลยหรือไม่ (หากเฉลยระบุไว้)
- **ตรวจสอบรูปแบบการจัดวางตัวบทกฎหมาย (Formatting):** ต้องมีการเว้นวรรคและย่อหน้าถูกต้องตามต้นฉบับ หากมีการรวมวรรค (Merged paragraphs) หรือจัดรูปแบบให้อ่านยาก ให้ถือว่าเป็นข้อบกพร่อง
- **ตรวจสอบความถูกต้องของถ้อยคำในตัวบทกฎหมาย (Text Accuracy):** ต้องยกมาตามตัวบทจริง ห้ามมีการตัดทอน แต่งเติม หรือสรุปความเอง หากไม่ตรงตามตัวบทจริง ให้หักคะแนน
- **การอ้างอิงมาตราเกิน:** หากมีการอ้างอิงมาตราเพิ่มเติมที่ไม่เกี่ยวข้อง (Irrelevant) และทำให้เกิดความสับสน ให้พิจารณาหักคะแนน (ต่างจากเดิมที่ไม่ถือว่าผิด)

# ผลการประเมิน
- อ้างอิงมาตรากฎหมายถูกต้องและครอบคลุมตามเฉลย
- อ้างอิงมาตรากฎหมายไม่ถูกต้อง หรือไม่เกี่ยวข้อง
- อ้างอิงมาตรากฎหมายถูกต้อง แต่ไม่ครอบคลุม มีมาตราสำคัญตกหล่น
