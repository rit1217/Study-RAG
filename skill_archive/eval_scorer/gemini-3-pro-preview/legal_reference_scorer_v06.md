# งานของคุณ
ให้คะแนนความถูกต้องของ **ข้อ 1 (มาตรากฎหมายที่เกี่ยวข้อง)** ในคำตอบ (OUTPUT) โดยเทียบกับเฉลย (EXPECTED)

**คำเตือนสำคัญ:** อย่าพิจารณาเพียงแค่เลขมาตราตรงกัน แต่ต้องอ่านเนื้อหาของตัวบทกฎหมายใน OUTPUT เทียบกับ EXPECTED ว่าใช้คำหรือสำนวนเดียวกัน และเว้นวรรคเหมือนกัน ถูกต้องทุกตัวอักษร

# รูปแบบข้อมูลเฉลย
เฉลย (EXPECTED) อยู่ในรูปแบบ Markdown ที่แบ่งเป็นหัวข้อ:
- `## 1. มาตรากฎหมายที่เกี่ยวข้อง` — มาตรากฎหมายและคำพิพากษาศาลฎีกาที่เกี่ยวข้อง (ใช้เป็นเฉลยหลักสำหรับการประเมินนี้)
- `## 2. คำวินิจฉัย` — คำวินิจฉัยที่ถูกต้อง
- `## 3. ข้อสรุปและข้อเสนอแนะ` — ข้อสรุปและข้อเสนอแนะ

ให้ประเมิน OUTPUT โดยเทียบกับ **ข้อ 1 (มาตรากฎหมายที่เกี่ยวข้อง)** ในเฉลย เป็นหลัก

# เกณฑ์การประเมิน
1. **ความถูกต้องของเนื้อหา (Verbatim Check):** 
   - ตรวจสอบเนื้อหาของมาตรากฎหมายแบบ **'คำต่อคำ'** 
   - ห้ามมีการตัดทอนเนื้อหา (เช่น การใช้เครื่องหมาย ...), การสรุปความเอง หรือการพิมพ์ผิด หากพบให้ถือว่าไม่ถูกต้องครบถ้วน (ให้คะแนน 0.5 หรือ 0.0 ตามความรุนแรง)
   - เลขมาตราและชื่อกฎหมายต้องถูกต้องแม่นยำ

2. **ความครอบคลุม:**
   - ตรวจสอบว่าครอบคลุมมาตรากฎหมายหลักทั้งหมดที่ระบุในเฉลยหรือไม่
   - ตรวจสอบว่ามีการอ้างอิงคำพิพากษาศาลฎีกาที่เกี่ยวข้องตามเฉลยหรือไม่ (หากเฉลยระบุไว้)

3. **ข้อมูลส่วนเกิน (Noise):**
   - หากมีการอ้างอิงมาตรากฎหมายที่ไม่เกี่ยวข้อง หรือเกินความจำเป็นจนทำให้เกิดความสับสน ให้พิจารณาตัดคะแนน (ต่างจากเดิมที่อนุโลมให้)

# ผลการประเมิน
- อ้างอิงมาตรากฎหมายไม่ถูกต้อง หรือไม่เกี่ยวข้อง (รวมถึงกรณีอ้างถูกมาตราแต่เนื้อหาผิดเพี้ยนไปมาก)
- อ้างอิงมาตรากฎหมายถูกต้อง แต่ไม่ครอบคลุม มีมาตราสำคัญตกหล่น (รวมถึงกรณีมีการตัดทอนเนื้อหา เว้นวรรคไม่ตรงกัน หรือสรุปความเอง)
- อ้างอิงมาตรากฎหมายถูกต้องและครอบคลุมตามเฉลย (เนื้อหาครบถ้วนสมบูรณ์ทุกตัวอักษร)
