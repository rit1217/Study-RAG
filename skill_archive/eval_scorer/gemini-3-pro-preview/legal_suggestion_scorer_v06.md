# งานของคุณ
ให้คะแนนความถูกต้องของ **ข้อ 3 (ข้อสรุปและข้อเสนอแนะ)** ในคำตอบ (OUTPUT) โดยเทียบกับเฉลย (EXPECTED)

# รูปแบบข้อมูลเฉลย
เฉลย (EXPECTED) อยู่ในรูปแบบ Markdown ที่แบ่งเป็นหัวข้อ:
- `## 1. มาตรากฎหมายที่เกี่ยวข้อง` — มาตรากฎหมายและคำพิพากษาศาลฎีกาที่เกี่ยวข้อง
- `## 2. คำวินิจฉัย` — คำวินิจฉัยที่ถูกต้อง
- `## 3. ข้อสรุปและข้อเสนอแนะ` — ข้อสรุปและข้อเสนอแนะ (ใช้เป็นเฉลยหลักสำหรับการประเมินนี้)

ให้ประเมิน OUTPUT โดยเทียบกับ **ข้อ 3 (ข้อสรุปและข้อเสนอแนะ)** ในเฉลย เป็นหลัก

# เกณฑ์การประเมิน
1. **ความเฉพาะเจาะจง (Specificity) และ Keyword:**
   - ข้อเสนอแนะต้องประกอบด้วยรายละเอียดที่เฉพาะเจาะจง (Specific Actionable Items) ตามที่ปรากฏในเฉลย เช่น จำนวนวัน (20 วัน), ชื่อหน่วยงานที่ต้องติดต่อ, หรือขั้นตอนทางกฎหมายที่ระบุชื่อเฉพาะ
   - หากเป็นคำแนะนำกว้างๆ (Generic) ที่ขาด Keyword สำคัญเหล่านี้ ให้ถือว่าไม่ครอบคลุม (ให้คะแนน 0.5)

2. **ความถูกต้องตามระเบียบ:**
   - ตรวจสอบว่าข้อเสนอแนะไม่ขัดต่อระเบียบปฏิบัติหรือเงื่อนไขที่ระบุไว้ในเฉลย แม้จะดูสมเหตุสมผลทั่วไปแต่ถ้าขัดกับเฉลยให้ถือว่าผิด

3. **การประเมินประโยชน์:**
   - ให้ประเมินประโยชน์ของข้อเสนอแนะแยกจากความถูกต้องของคำวินิจฉัยในระดับหนึ่ง หากคำวินิจฉัยผิดพลาดเล็กน้อย แต่ข้อเสนอแนะมีขั้นตอนปฏิบัติที่ถูกต้องและเป็นประโยชน์ตามเฉลย ให้พิจารณาคะแนนตามความเหมาะสมของเนื้อหาข้อเสนอแนะนั้น

4. **สำคัญ: ความครอบคลุมของข้อเสนอแนะ**
  - ตรวจสอบว่าสามารถให้ข้อเสนอแนะได้ครบตามที่ระบุไว้ในเฉลย และไปในทิศทางเดียวกัน

5. **ความเกี่ยวข้องกับคำวินิจฉัย**
   - การสรุปและเสนอแนะ จะต้องเป็นไปในทิศทางเดียวกับคำวินิจฉัย

# ผลการประเมิน
- ไม่ถูกต้องตามเฉลย (ขัดแย้งกับเฉลย หรือให้คำแนะนำที่ผิดระเบียบ)
- ตอบได้ถูกต้อง แต่ไม่ครอบคลุม มีเนื้อหาสำคัญตกหล่น (คำแนะนำกว้างเกินไป คำแนะนำตกหล่นจากเฉลย ขาดความเฉพาะเจาะจง มีการสรุปนอกเหนือจากคำวินิจฉัย)
- ตอบได้ถูกต้อง ชัดเจน ครอบคลุม ให้ข้อเสนอแนะที่เป็นไปได้จริง (มี Keyword ครบถ้วน)
