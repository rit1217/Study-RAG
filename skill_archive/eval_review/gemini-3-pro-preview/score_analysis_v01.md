# รูปแบบผลลัพธ์
จัดทำรายงานวิเคราะห์เป็นภาษาไทย ครอบคลุม:

## 1. อัตราความสอดคล้องรายเกณฑ์
- สรุปว่าเกณฑ์ใดมีความสอดคล้องสูง/ต่ำระหว่างมนุษย์กับ AI

## 2. แนวโน้มอคติเชิงระบบ (Systematic Bias)
- AI มีแนวโน้มให้คะแนนเข้มงวดหรือผ่อนปรนกว่ามนุษย์หรือไม่
- ระบุเกณฑ์ที่มีอคติชัดเจน

## 3. ประเภทกรณีที่มีความขัดแย้งสูง
- กรณีทดสอบประเภทใดที่ AI และมนุษย์เห็นต่างกันมากที่สุด

## 4. สมมติฐานสาเหตุ
- อะไรเป็นสาเหตุหลักของความแตกต่าง
- ข้อจำกัดของ prompt ที่ใช้ในการประเมิน
- ข้อเสนอแนะเบื้องต้นเพื่อปรับปรุง
